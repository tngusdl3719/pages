Base64 Encoder / Decoder

문자열을 Base64로 인코딩하거나 Base64 문자열을 다시 UTF-8 텍스트로 복원합니다. 파일 업로드 없이 브라우저에서만 처리합니다.

Tool UI

바로 사용

사용 예시

  1. hello world를 넣고 인코딩하면 aGVsbG8gd29ybGQ=가 됩니다.
  2. 같은 값을 다시 넣고 디코딩하면 원문으로 복원됩니다.

사용 방법

  1. 문자열 또는 Base64 값을 입력 영역에 넣습니다.
  2. 인코딩 또는 디코딩을 눌러 결과를 확인합니다.
  3. 필요하면 결과 복사 버튼으로 가져갑니다.

자주 묻는 질문

  • 처리는 모두 브라우저 안에서 이뤄집니다.
  • UTF-8 기준이라 한글도 인코딩할 수 있습니다.
  • 잘못된 Base64는 오류 상태로 안내합니다.
