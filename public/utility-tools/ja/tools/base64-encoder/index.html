Base64 Encoder / Decoder

通常の文字列をBase64へ変換し、Base64文字列をUTF-8テキストへ戻します。 すべての処理はブラウザで完結し、バックエンドは使いません。

ホーム ツール一覧 Base64

Tool UI

利用エリア

使用例

  1. 入力: hello world
  2. 出力: aGVsbG8gd29ybGQ=
  3. デコード: 出力値を戻せば元の文字列になります。

使い方

  1. 文字列またはBase64値を入力欄に貼り付けます。
  2. 目的に応じてエンコードまたはデコードを押します。
  3. 必要なら結果をコピーして次の作業に使います。

よくある用途

  • エンコード済みペイロードの確認
  • テキストの簡易変換
  • デバッグ中のBase64文字列確認

入力内容は送信されますか?

いいえ。処理はブラウザ内だけで完結します。

日本語も変換できますか?

UTF-8を前提にしているため、日本語や韓国語も扱えます。

無効なBase64を入れるとどうなりますか?

エラー状態へ切り替え、入力値の確認を促すメッセージを表示します。

URL Encoder / Decoder

JWT Decoder

トークン確認フローでBase64と合わせて使いやすい補助ツールです。

Random String Generator

テスト用の文字列をすばやく作る際に役立ちます。
