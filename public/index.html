Cloudflare Pages 설정값

Git 연동 배포

  • Framework preset: None
  • Build command: 비워두기
  • Build output directory: public

직접 업로드 배포

  • public/ 폴더를 그대로 업로드
  • HTML, CSS 파일만으로 서비스 가능
  • 추가 런타임이나 서버 설정 불필요
